## Lesson3:

### Task1 - Полная загрузка данных

full_load.sql - загрузка данных через values и через select с другой таблицы,
также сюда относится первое занесение данных с API из Task2

### Task2 - Инкрементная загрузка

Три скрипта:

incremental_load.py - сбор данных по API, для получения температуры в заданных городах,
данные заносятся в БД PotgreSQL новыми строчками в таблицу cityes_2.

incremental_load_2.py - новые города заносятся новыми строчками,
у ранее внесённых городов обновляются данные (температура и дата)
в таблице cityes_weather_update.

cdc_delta.sql - инкрементная загрузка с выделением дельты
с триггером и таблицей логов (журнал изменений)

Папка airflow_autoload - загрузка через Airflow
данных погоды с API раз в час.
Даг с кастомным оператором и скриптом для него (плагином).

### Task3 - Real-Time Loading

Потоковая загрузка актуальной цены акции SBER
из API московской биржи MOEX

producer.py - забирает данные из API MOEX раз в 1 сек,
если цена изменилась, то грузит её в RabbitMQ в формате json
(напр: {"SBER": 313.61})

consumer.py - клиент, как только есть данные в очереди API_SBER
в RabbitMQ, забирает их от туда и грузит в БД PostgreSQL
